The Library Management System is a console-based application developed using C++ and SQLite3 that helps manage library operations efficiently. The system provides role-based access for Managers, Librarians, and Members, allowing different users to perform specific tasks according to their permissions. It supports essential library functions such as user authentication, book registration, book searching, issuing and returning books, user management, and transaction tracking.

The system stores all data in an SQLite database, including information about users, books, and borrowing transactions. Managers can register and delete users, librarians can manage books and transactions, while members can search and view available books. The system also includes features such as borrow limits, automatic due-date generation, availability checking, and recent activity monitoring.

This project is designed to demonstrate database integration with C++, file and data management concepts, role-based system design, and basic library automation principles.
